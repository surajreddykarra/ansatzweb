# Assets

Use this folder for homepage media.

- `assets/logos/` for studio and product logos
- `assets/zync/` for Zync screenshots and promo images
- `assets/backgrounds/` for textures, gradients, or decorative visuals

The homepage currently reads:

- `assets/logos/ansatz-logo.png`
- `assets/logos/zync-logo.svg`
- `assets/zync/zync-intro.png`

You can drop in more screenshots and extend the carousel markup in `index.html` without changing the overall page layout.
